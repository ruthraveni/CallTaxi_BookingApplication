
This project implements a Call Taxi Booking system using Java and OOPS concepts.
Taxis are automatically assigned based on availability, shortest distance, and lowest earnings.
The system calculates fare dynamically based on travel distance between route points.
It also tracks booking history and total earnings for each taxi in a structured way.
